    1.Block = vem como padrão: div/ p/ h1 h2 h3 h4 h5 h6/ ul/ li / article

  • -sempre ocupa toda sua linha

  • -Podemos definir o width e o height
  • -Podemos definir o valores das margin

    2.Inline = vem por padrão a, span, b, i:
  • -Elementos em linha
  • -Nao permitir alterar o width e height
  • -Nao podemos definir o valores de margin top e bottom
  • -Ganha comportamento de uma palavra
minha div inline 1
minha div inline 2
minha div inline 3


    Inline vai ignora os valores de width, height, margin-top e margin-bottom
